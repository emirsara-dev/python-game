# My Python Game

A tiny test game written in Python.

How to play:
The one who picks the last stone loses :)

## Quick start

1. Make sure you have Python 3 installed.  
2. Run the game:

```bash
python main.py
```
## Notes

main.py is the entry point.

stones.py contains supporting game logic.
